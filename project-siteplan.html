Fitness for Beginners - Site Plan

Site Name

FitStart - This name reflects the purpose of helping people begin their fitness journey with motivation and simple routines.

Site Purpose

This site helps beginners start their fitness journey at home. It includes illustrated workout routines, habit-building tips, and healthy lifestyle guidance.

Scenarios

  • What are some simple workout routines I can start at home with no equipment?
  • How do I start eating healthier without following a strict diet?

Color Schema

Background: #f8f7f7

Primary: #a10e0d (headings, buttons)

Accent: #808080 (subtext, borders)

Text/Dark: #171818 (body text, footer)

Typography

  • Headings: Oswald, sans-serif (light, bold style)
  • Body text: Open Sans, sans-serif (clean and readable)

Wireframes

Sketches will show layout for mobile and desktop home pages, including logo, nav, hero section, and key content areas.

  • Mobile View: Logo, hamburger menu, intro section, list of routines
  • Desktop View: Horizontal nav, hero banner, routines in columns or grid, footer
